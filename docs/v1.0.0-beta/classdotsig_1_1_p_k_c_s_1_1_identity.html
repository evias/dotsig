Detailed Description

Class template for PKCS (RSA) identities that consist of a keypair.

This class template specializes the ParentType with Botan::RSA_PrivateKey and Botan::RSA_PublicKey. This implementation can be used to generate, import and/or export PKCS1 (RSA) keys, as well as to sign messages or files digitally using the PKCS1 v1.5 standard for digital signatures.
