Detailed Description

template<class PrivateKeyImpl, class PublicKeyImpl>
class dotsig::Identity< PrivateKeyImpl, PublicKeyImpl >

Template class for identities that consist of a private/public keypair.

This template class accepts two template parameters with PrivateKeyImpl and PublicKeyImpl. Importantly, the current implementation of the copy constructor makes it so that these template parameters must be compatible with Botan's pk_keys.h format. Method implementations must be provided by these classes for the methods: algorithm_identifier, private_key_bits and public_key_bits, as found in Botan::Private_Key and Botan::Public_Key.
