Detailed Description

A class that implements the factory pattern for identities.

This (minimal) factory pattern implementation deals with IIdentity objects and a uses a pointer creation function to create pointers of Identity classes that are supported by the software. These creator functions may or may not prepare the object before it is returned, as deemed necessary.
