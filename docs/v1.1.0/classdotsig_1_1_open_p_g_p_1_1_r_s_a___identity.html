Detailed Description

Class for OpenPGP identities using type RSA (PKCS1 v1.5).

This class template specializes the OpenPGP Identity template class with Botan::RSA_PrivateKey and Botan::RSA_PublicKey.

Note
This identity wrapper exports BER-encoded private keys to the user's home folder, in a file named id_openpgp and it exports PEM-encoded public keys to the user's home folder, in a file named id_openpgp.pub.

dotsig::OpenPGP::RSA_Identity::RSA_Identity ( )
inline

Default constructor. Creates an empty OpenPGP RSA (PKCS1 v1.5) identity.

This class template specifies a padding scheme and a hash function, with the resulting parameters specification being: PKCS1v15(SHA-256).

Note
The RSA implementation requires both a padding scheme and hash function.
The created identity does not have a private key, make sure to use one of GenerateRandom or Import to populate it.
